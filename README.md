# 🍴 The Golden Fork - Premium Restaurant & Food Delivery

Welcome to **The Golden Fork**, a high-end, production-ready food delivery web application built with **Next.js 15**, **Clerk**, and **Hygraph**. This application offers a seamless dining experience from browsing a curated menu to a secure checkout.

## 🚀 Live Demo
Check out the live application here:
**[https://the-golden-fork-pz96.vercel.app/](https://the-golden-fork-pz96.vercel.app/)**

## ✨ Key Features
- **Modern UI/UX**: A sleek, responsive design optimized for all devices.
- **Secure Authentication**: User management powered by **Clerk**.
- **Dynamic Menu**: Real-time content management via **Hygraph (GraphQL)**.
- **Smart Cart System**: Persistent cart data with coupon and discount support.
- **Optimized Checkout**: streamlined flow with **Cash on Delivery (COD)** support.
- **Performance First**: Built with Next.js 15/16 and optimized for fast page loads and SEO.

## 🛠️ Technology Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: Tailwind CSS
- **Authentication**: [Clerk](https://clerk.com/)
- **Backend/CMS**: [Hygraph](https://hygraph.com/) (GraphQL)
- **Deployment**: [Vercel](https://vercel.com/)
- **Icons**: Lucide React
- **Notifications**: React Hot Toast

## 📦 Getting Started

### Prerequisites
Ensure you have Node.js installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shreyaaaah/TheGoldenFork.git
   ```
2. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
3. Set up environment variables:
   Create a `.env` file and add your Clerk and Hygraph credentials.

4. Run the development server:
   ```bash
   npm run dev
   ```
